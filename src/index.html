Конструктор email-шаблонів для команди

Форматування HTML/MJML, обробка зображень, тестова розсилка, бібліотеки блоків і шаблонів — в одному застосунку. Ця сторінка допомагає команді ознайомитись з проєктом і оцінити, які фічі реально потрібні.

Що вже є в застосунку

10 модулів, зібраних з CLAUDE.md проєкту. Два з них наразі вимкнені фіче-флагами.

🧩

HTML Converter

Перетворює текст/HTML у email-safe HTML і MJML, з готовими шаблонами та експортом.

🛠️

Advanced Converter

DOM-пайплайн для вставки з Google Docs: preprocess → normalize → IR → classify → render.

🖼️

Image Converter

Клієнтська конвертація зображень (JPEG/PNG/WebP/AVIF) на WASM, без сервера.

✉️

Email Sender

Надсилання тестових листів через SMTP (Gmail/Outlook/Yahoo) з формою креденшлів.

📚 🚧 вимкнено

Block Library

Бібліотека email-блоків з категоріями, попереднім переглядом і кастомними блоками.

🗂️

Template Library

Керування шаблонами листів через файлову систему з пошуком і синхронізацією.

✅ 🚧 вимкнено

Email Validator

Перевірка HTML на сумісність з email-клієнтами: теги, CSS, доступність.

🤖

AI ALT-текст

Генерація ALT-тексту і назв зображень через OCR + CLIP + Gemma 3 (локальний Ollama).

🚀

Storage Automation

Playwright-автоматизація завантаження файлів у сховища (default/alphaone/ttt).

💻

Desktop (Electron)

Пакування застосунку в десктопний клієнт для macOS і Windows.

📘

Повний гайд — скоро

Покроковий розбір кожного модуля зі скрінами. Наразі в процесі написання.
